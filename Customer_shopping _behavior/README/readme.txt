# 📊 Customer Behaviour Analysis Dashboard

## 🚀 Project Overview

This project analyzes customer shopping behavior and purchasing patterns using Python, MySQL, and Power BI.

The objective is to transform raw customer transaction data into actionable business insights through data cleaning, exploratory data analysis (EDA), and interactive visualizations.

---

## 🛠️ Tech Stack

- Python
- Pandas
- Google Colab
- MySQL Workbench
- Power BI
- Git & GitHub

---

## 📂 Project Workflow

### 1️⃣ Data Cleaning
Performed data preprocessing using Pandas in Google Colab:

- Checked missing values
- Removed duplicate records
- Standardized column formats
- Corrected inconsistent data entries
- Exported cleaned dataset

### 2️⃣ Exploratory Data Analysis (EDA)
Used MySQL Workbench to perform analysis such as:

- Customer distribution
- Revenue by category
- Sales by category
- Revenue by age group
- Sales by age group
- Subscription behavior analysis
- Review rating analysis

### 3️⃣ Dashboard Development
Built an interactive Power BI dashboard featuring:

- Customer Count KPI
- Average Purchase Amount KPI
- Average Review Rating KPI
- Subscription Distribution
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group

Interactive Filters:
- Subscription Status
- Gender
- Product Category
- Shipping Type

---

## 📈 Key Insights

### Customer Insights
- Majority of customers are non-subscribers.
- Average purchase amount is around $60.
- Average review rating remains above 3.7.

### Category Performance
- Clothing generates the highest revenue and sales.
- Accessories contribute significantly to total revenue.
- Outerwear records the lowest sales performance.

### Age Group Analysis
- Young Adults contribute the highest revenue.
- Middle-aged customers remain a strong purchasing segment.
- Senior customers contribute comparatively less.
